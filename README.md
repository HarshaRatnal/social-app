# social-app
## Dependencies

- NodeJS, ExpressJS
- MongoDB

## Installation

```bash
$ npm install
```
### `config` environments

PORT: 5000

JWT_SECRET: 

MONGO_URL: localhost

MONGO_PORT: 27017

MONGO_DB: social-app

MONGO_URI: mongodb://localhost:27017/social-app

- Create .env and use these above variables

## Running the app

```bash
$ "dev": "nodemon index.js"
$ npm run dev
```
